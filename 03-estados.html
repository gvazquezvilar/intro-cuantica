Introducción a la comunicación y la computación cuántica

3 Estados cuánticos binarios

En la sección anterior hemos planteado el problema de cómo modelar la polarización de un fotón y hemos introducido una curiosa notación para referirnos a los vectores. En ésta y en las siguientes secciones introduciremos un modelo matemático general para describir un sistema cuántico. Este modelo permite describir la polarización de un fotón, la medida del spin de un electrón, así como a muchos otros sistemas con propiedades cuánticas. Por tanto, gracias a este modelo podremos abstraernos de la tecnología cuántica utilizada.

Para caracterizar el proceso de medida de un sistema con propiedades cuánticas, necesitamos modelar el estado cuántico, caracterizar matemáticamente las observaciones fruto de esta medida, y determinar el efecto de esta medida sobre el estado original.

3.1 Estados cuánticos puros

Utilizando la notación bra-ket, definimos los vectores ∣0⟩=[10],∣1⟩=[01],\begin{align*} \ket{0}= \biggl[\begin{matrix}1\\0\end{matrix}\biggr],\quad \ket{1}= \biggl[\begin{matrix}0\\1\end{matrix}\biggr], \end{align*}∣0⟩=[10​],∣1⟩=[01​],​ que se corresponden con la base canónica del espacio de Hilbert de dimensión 222.

Un estado binario puro general se puede escribir como ∣ψ⟩=α∣0⟩+β∣1⟩.\begin{align*} \ket{\psi} &= \alpha \ket{0} + \beta \ket{1}. \end{align*}∣ψ⟩​=α∣0⟩+β∣1⟩.​ Los parámetros α\alphaα y β\betaβ son dos números complejos que se denominan amplitudes de probabilidad, que veremos que nos permitirán calcular las probabilidades de medida de este sistema. Estas amplitudes de probabilidad deben cumplir una cierta normalización y se tiene que ∣α∣2+∣β∣2=1|\alpha|^2 + |\beta|^2 = 1∣α∣2+∣β∣2=1.

Este estado se conoce habitualmente como cúbit (o en inglés, qubit) y se corresponde al estado fundamental de la mecánica cuántica. En esta representación surge también el concepto de superposición, que indica que este sistema no se encuentra en el estado ∣0⟩\ket{0}∣0⟩ o ∣1⟩\ket{1}∣1⟩, sino en una superposición de ambos. Veremos este concepto en detalle al analizar el proceso de medida del sistema.

Ejemplo 3.1 El modelo de un fotón polarizado linealmente con un ángulo θ\thetaθ se corresponde con un cúbit ya que para α=cos⁡(θ)\alpha=\cos(\theta)α=cos(θ) y β=sin⁡(θ)\beta=\sin(\theta)β=sin(θ), se tiene que ∣ψ⟩=cos⁡(θ)∣0⟩+sin⁡(θ)∣1⟩=[cos⁡(θ)sin⁡(θ)].\begin{align*} \ket{\psi} &= \cos(\theta) \ket{0} + \sin(\theta) \ket{1} = \begin{bmatrix}\cos(\theta)\\ \sin(\theta)\end{bmatrix}. \end{align*}∣ψ⟩​=cos(θ)∣0⟩+sin(θ)∣1⟩=[cos(θ)sin(θ)​].​ Así, un fotón polarizado se puede interpretar como un vector con corrdenadas cos⁡(θ)\cos(\theta)cos(θ) y sin⁡(θ)\sin(\theta)sin(θ) en una base formada por los estados ortogonales ∣0⟩\ket{0}∣0⟩ y ∣1⟩\ket{1}∣1⟩, tal y como representamos en la siguiente figura:

Los estados cuánticos puros se corresponden con sistemas cuánticos que se encuentran en un estado definido, aunque como veremos, su medida puede producir resultados aleatorios. Para modelar sistemas los que no conocemos su estado de forma precisa, o que presentan cierta incertidumbre no asociada a la superposición, es necesario definir un modelo más general que presentamos a continuación.

3.2 Matriz de densidad de probabilidad

De forma general, vamos a describir el estado de un sistema cuántico con incertidumbre por medio de la denominada matriz de densidad de probabilidad. Para un estado cuántico binario esta matriz tiene dimensión 2×22 \times 22×2 y está dada por: ρ=[ρ11ρ12ρ21ρ22].\begin{align*} \rho = \biggl[\begin{matrix}\rho_{11} & \rho_{12}\\ \rho_{21} & \rho_{22}\end{matrix}\biggr]. \end{align*}ρ=[ρ11​ρ21​​ρ12​ρ22​​].​ Una matriz de densidad de probabilidad debe cumplir las siguientes propiedades:

  • Semidefinida positiva, ρ⪰0\rho\succeq 0ρ⪰0.
  • Auto-hermítica (a veces denominada simplemente hermítica), ρ=ρH\rho=\rho^Hρ=ρH.
  • Traza unitaria, Tr[ρ]=1\text{Tr}[\rho] = 1Tr[ρ]=1.

Rigurosamente, un estado cuántico binario puro se define como un operador acotado en un espacio de Hilbert de dimensión 222, y las matrices de densidad son una combinación probabilistica de estos estados. A efectos de este curso, simplemente identificaremos un estado cuántico con una matriz ρ\rhoρ con las citadas propiedades.

Tipos de estados cuánticos

  • Estado puro: Una matriz de densidad de probabilidad ρ\rhoρ describe un estado puro si el rango de ρ\rhoρ es 111. La matriz de densidad de probabilidad de un estado puro ∣ψ⟩\ket{\psi}∣ψ⟩, está dada por ρ=∣ψ⟩⟨ψ∣\rho = \ket{\psi}\bra{\psi}ρ=∣ψ⟩⟨ψ∣.

  • Estado mixto: Decimos que un estado ρ\rhoρ es mixto si no es puro, es decir, si el rango de ρ\rhoρ es mayor que 111. La matriz de densidad de probabilidad de un estado mixto se corresponde a una mezcla probabilística de un conjunto de estados puros {∣ψx⟩}\bigl\{ \ket{\psi_x} \bigr\}{∣ψx​⟩}: ρ=∑x∈XpX(x)∣ψx⟩⟨ψx∣.\begin{align*} \rho = \sum_{x\in\mathcal{X}} p_X(x) \ket{\psi_x} \bra{\psi_x}. \end{align*}ρ=x∈X∑​pX​(x)∣ψx​⟩⟨ψx​∣.​

  • Estado clásico: Consideramos que un estado ρ\rhoρ es clásico si es una matriz diagonal. Este estado representa una variable aleatoria binaria con probabilidades dadas por los elementos de la diagonal de ρ\rhoρ. Nótese que un estado clásico puede ser también puro o mixto.

Explicaremos en detalle estos tipos de estados con varios ejemplos concretos en las próximas secciones.

3.3 Modelando un bit aleatorio

En primer lugar, vamos a tomar un bit clásico que toma los valores ‘0’ y ‘1’. Se puede asociar cada uno de estos valores a los dos vectores de la base canónica [10]\bigl[\begin{smallmatrix}1\\0\end{smallmatrix}\bigr][10​] y [01]\bigl[\begin{smallmatrix}0\\1\end{smallmatrix}\bigr][01​]. Utilizando la notación bra-ket, tenemos que ⟨0∣=∣0⟩H=[10],⟨1∣=∣1⟩H=[01].\begin{align*} %\ket{0}&= \biggl[\begin{matrix}1\\0\end{matrix}\biggr] \quad\Rightarrow\quad \bra{0} = \ket{0}^H = [\begin{matrix}1 &0\end{matrix}],\\ %\ket{1}&= \biggl[\begin{matrix}0\\1\end{matrix}\biggr] \quad\Rightarrow\quad \bra{1}= \ket{1}^H = [\begin{matrix}0 &1\end{matrix}]. \end{align*}⟨0∣=∣0⟩H=[1​0​],⟨1∣=∣1⟩H=[0​1​].​

Bit determinista

Para un bit clásico en el estado ‘0’, definimos su matriz de densidad de probabilidad asociada como ρ0=∣0⟩⟨0∣=[1000].\begin{align*} \rho_0 = \ket{0} \bra{0} = \biggl[\begin{matrix}1 & 0\\ 0 & 0\end{matrix}\biggr]. \end{align*}ρ0​=∣0⟩⟨0∣=[10​00​].​ De forma análoga, para un bit en estado ‘1’, su matriz de densidad de probabilidad está dada por ρ1=∣1⟩⟨1∣=[0001].\begin{align*} \rho_1 = \ket{1} \bra{1} = \biggl[\begin{matrix}0 & 0\\ 0 & 1\end{matrix}\biggr]. \end{align*}ρ1​=∣1⟩⟨1∣=[00​01​].​ Por tanto, para un bit clásico ‘0’ o ‘1’ sus matrices de densidad asociadas se corresponden con un estado puro, ya que tienen rango 111, pero al mismo tiempo definen un estado clásico, ya que son matrices diagonales.

Variable aleatoria binaria

Consideremos ahora una variable aleatoria binaria X∈{0,1}X\in\{0,1\}X∈{0,1} que sigue una distribución de probabilidad Bernoulli con parámetro ppp, es decir PX(1)=pP_X(1) = pPX​(1)=p y PX(0)=1−pP_X(0) = 1-pPX​(0)=1−p. La matriz de densidad de probabilidad de XXX se corresponde a la mezcla de las matrices ρ0\rho_0ρ0​ y ρ1\rho_1ρ1​ definidas anteriormente: ρX=(1−p)ρ0+pρ1=[1−p00p].\begin{align*} \rho_X = (1-p) \rho_0 + p \rho_1 = \biggl[\begin{matrix}1-p & 0\\ 0 & p\end{matrix}\biggr]. \end{align*}ρX​=(1−p)ρ0​+pρ1​=[1−p0​0p​].​ Así, una variable aleatoria tambien se puede representar como una matriz de densidad de probabilidad. En general, una matriz de densidad de probabilidad diagonal se corresponde con un estado clásico y la correspondiente distribución de probabilidad se encuentra en la diagonal de la matriz. Para 0<p<10<p<10<p<1, se tiene que este estado es un estado mixto, ya que el rango de la matriz es igual a 222.

Ejemplo 3.2 Consideremos el experimento de lanzar una moneda no trucada, y ocultar su resultado. Este experimento se corresponde con una distribución de probabilidad Bernoulli con parámetro p=1/2p= 1/2p=1/2. La matriz de densidad de probabilidad asociada a este experimento está dada por ρX=[1/2001/2].\begin{align*} \rho_X = \biggl[\begin{matrix}1/2 & 0\\ 0 & 1/2\end{matrix}\biggr]. \end{align*}ρX​=[1/20​01/2​].​ Este es un estado mixto, ya que ρX\rho_XρX​ tiene rango 2>12 > 12>1, y clásico, al ser ρX\rho_XρX​ una matriz diagonal.

3.4 Modelando un fotón polarizado

Vamos a estudiar ahora el caso más interesante de cómo modelar un fotón por medio de una matriz de densidad de probabilidad. Éste es un sistema cuántico y, por tanto, no se puede modelar utilizando únicamente probabilidad clásica.

Un fotón polarizado linealmente

Un fotón polarizado linealmente con ángulo θ\thetaθ se asocia al estado ∣θ⟩=[cos⁡(θ)sin⁡(θ)]\ket{\theta} = \bigl[\begin{smallmatrix}\cos(\theta)\\ \sin(\theta)\end{smallmatrix}\bigr]∣θ⟩=[cos(θ)sin(θ)​]. Definimos la matriz de densidad de probabilidad asociada a ese estado como ρ=∣θ⟩⟨θ∣=[cos⁡(θ)2cos⁡(θ)sin⁡(θ)cos⁡(θ)sin⁡(θ)sin⁡(θ)2].\begin{align*} \rho = \ket{\theta} \bra{\theta} = \biggl[\begin{matrix} \cos(\theta)^2 & \cos(\theta)\sin(\theta)\\ \cos(\theta)\sin(\theta) & \sin(\theta)^2 \end{matrix}\biggr]. \end{align*}ρ=∣θ⟩⟨θ∣=[cos(θ)2cos(θ)sin(θ)​cos(θ)sin(θ)sin(θ)2​].​ Se puede observar, que incluso si escogemos p=sin⁡(θ)2=1−cos⁡(θ)2p = \sin(\theta)^2 = 1- \cos(\theta)^2p=sin(θ)2=1−cos(θ)2, esta matriz no coincide con la del estado ρX\rho_XρX​ definido en la sección anterior para una variable aleatoria clásica. Esto es porque una variable aleatoria clásica es fundamentalmente diferente de un sistema con propiedades cuánticas, incluso si tienen los mismos valores en la diagonal de su matriz de densidad.

Un fotón con incertidumbre

El paradigma que hemos estudiado permite modelar tanto un sistema que presenta incertidumbre (como una variable aleatoria clásica), como un sistema cuántico en un estado definido, tal y como puede ser un fotón con una polarización dada. Además, este modelo matemático permite modelar una combinación de estos dos efectos de forma simultánea, como muestra el siguiente ejemplo.

Ejemplo 3.3 Modelamos un fotón con polarización lineal θ\thetaθ con el estado puro ∣θ⟩=[cos⁡(θ)sin⁡(θ)]\ket{\theta} = \bigl[\begin{smallmatrix}\cos(\theta)\\ \sin(\theta)\end{smallmatrix}\bigr]∣θ⟩=[cos(θ)sin(θ)​]. Consideremos un experimento que genera un fotón con una polarización bien ∣π/3⟩\ket{\pi/3}∣π/3⟩ o ∣π/4⟩\ket{\pi/4}∣π/4⟩, pero desconocemos cuál de ellas. Si las dos polarizaciones tienen la misma probabilidad, el estado resultante se corresponde con la mezcla probabilística de las matrices de densidad de cada posibilidad: ρ=12∣π/3⟩⟨π/3∣+12∣π/4⟩⟨π/4∣.\begin{align*} \rho = \frac{1}{2} \ket{\pi/3} \bra{\pi/3} + \frac{1}{2} \ket{\pi/4} \bra{\pi/4}. \end{align*}ρ=21​∣π/3⟩⟨π/3∣+21​∣π/4⟩⟨π/4∣.​ Se puede comprobar que ρ\rhoρ cumple las propiedades de una matriz de densidad de un sistema cuántico: ρ⪰0\rho\succeq 0ρ⪰0, ρ=ρH\rho=\rho^Hρ=ρH y Tr[ρ]=1\text{Tr}[\rho]=1Tr[ρ]=1. Además en este caso, el estado resultante es un estado mixto, ya que el rango de ρ\rhoρ es igual a 2>12 > 12>1.
